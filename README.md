## issue

我在使用pyecharts的set_global_opts的时候发现visualmap_opts，中用prices指定上下限的范围会发声浮动。
具体来说例如：`"min": 1, "max": 49`此时运行,从结果上来看所指范围是`"1 or 2 ~ 49 or 50 or 51"`这个bug在有些图里出现有些又没有.
这个bug是我把运行结果用 `simplejson` 转换成json再去读取它之后出现的.
